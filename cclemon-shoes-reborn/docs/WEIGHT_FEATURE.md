# 體重管理功能說明

本文檔說明 CCLEMON-UI 中新增的體重管理功能。

## 功能概述

體重管理功能包含兩個主要頁面：
1. **體重記錄頁面** (`/weight`) - 用於記錄和查看體重數據
2. **體重趨勢頁面** (`/weight/chart`) - 用於查看體重變化圖表和統計資訊

## API 端點

後端 API 基於 `/api/v1/weights`，提供以下功能：

### 1. 記錄/更新體重
- **端點**: `POST /api/v1/weights`
- **說明**: 記錄新的體重數據或更新同一天的體重記錄
- **請求參數**:
  ```typescript
  {
    measureDate?: string;  // 測量日期 (YYYY-MM-DD)，選填，預設為今天
    weightKg: number;      // 體重 (公斤)，必填，範圍: 20-300
    measureTime?: string;  // 測量時間 (HH:mm:ss)，選填
    note?: string;        // 備註，選填
  }
  ```

### 2. 查詢體重記錄列表
- **端點**: `GET /api/v1/weights`
- **說明**: 分頁查詢體重記錄列表
- **查詢參數**:
  - `startDate`: 開始日期 (YYYY-MM-DD)，選填
  - `endDate`: 結束日期 (YYYY-MM-DD)，選填
  - `page`: 頁碼 (從 0 開始)，預設 0
  - `size`: 每頁數量，預設 30

### 3. 獲取體重圖表數據
- **端點**: `GET /api/v1/weights/chart`
- **說明**: 獲取指定時間範圍的體重趨勢數據
- **查詢參數**:
  - `range`: 時間範圍，可選值: `7d`, `30d`, `90d`, `custom`
  - `startDate`: 自訂開始日期 (當 range=custom 時必填)
  - `endDate`: 自訂結束日期 (當 range=custom 時必填)

### 4. 刪除體重記錄
- **端點**: `DELETE /api/v1/weights/{id}`
- **說明**: 刪除指定的體重記錄

## 頁面功能

### 體重記錄頁面 (/weight)

#### 功能特性：
1. **新增體重記錄**
   - 日期選擇器 (預設為今天)
   - 體重輸入 (公斤，範圍 20-300)
   - 時間選擇器 (選填)
   - 備註欄位 (選填)

2. **查看歷史記錄**
   - 日期範圍篩選
   - 分頁顯示
   - 每筆記錄顯示：日期、時間、體重、備註

3. **刪除記錄**
   - 點擊刪除按鈕
   - 確認對話框

#### 驗證規則：
- 體重必須在 20-300 公斤之間
- 測量日期不能是未來日期
- 日期和時間格式必須正確

### 體重趨勢頁面 (/weight/chart)

#### 功能特性：
1. **時間範圍選擇**
   - 快速選擇：7天、30天、90天
   - 自訂範圍：可選擇任意開始和結束日期

2. **統計資訊卡片**
   - 平均體重
   - 最高體重
   - 最低體重
   - 體重變化 (首尾差異)

3. **趨勢圖表**
   - 折線圖顯示體重變化
   - 自動計算 Y 軸範圍
   - 顯示數據點和數值
   - X 軸顯示日期標籤

## 導航

在側邊欄中添加了兩個新的導航項目：
- **體重記錄** - 導航到記錄頁面
- **體重趨勢** - 導航到圖表頁面

## 技術實現

### 前端技術棧
- Vue 3 (Composition API)
- Quasar Framework
- TypeScript
- Axios (API 調用)
- Canvas (圖表繪製)

### 文件結構
```
cclemon-ui/src/
├── api/
│   └── weight/
│       ├── types.ts          # TypeScript 類型定義
│       └── weight.ts         # API 調用方法
├── pages/
│   ├── WeightPage.vue        # 體重記錄頁面
│   └── WeightChartPage.vue   # 體重趨勢頁面
└── router/
    └── routes.ts             # 路由配置 (已更新)
```

### API 配置
使用 `healthApi` 實例進行 API 調用，基礎 URL 配置在環境變數 `VUE_APP_HEALTH_BASE_URL` 中。

## 使用流程

1. **記錄體重**
   - 打開側邊欄，點擊「體重記錄」
   - 填寫表單 (日期、體重等)
   - 點擊「提交」按鈕
   - 成功後會自動刷新列表

2. **查看歷史記錄**
   - 在體重記錄頁面下方查看列表
   - 可使用日期範圍篩選器
   - 支援分頁瀏覽

3. **查看趨勢圖表**
   - 打開側邊欄，點擊「體重趨勢」
   - 選擇時間範圍 (7天/30天/90天/自訂)
   - 查看統計資訊和趨勢圖表

4. **刪除記錄**
   - 在記錄列表中找到要刪除的項目
   - 點擊刪除按鈕
   - 確認刪除操作
